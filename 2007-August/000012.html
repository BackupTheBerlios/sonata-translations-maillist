"INTEGER" and "EXPRESSION" will be used directly in gettext library.
INTEGER must be a decimal number which specifies how many different
plural forms exist for this language. EXPRESSION is an expression
which is using the C language syntax. The only variable allowed in
EXPRESSION is n. For example, in English, since the singular form is
used for one only, this line would be:
  Plural-Forms: nplurals=2; plural=n != 1;
But in French, this line would be:
  Plural-Forms: nplurals=2; plural=n > 1;
since the singular form is used for zero and one.
