On 10/27/07, Alex <alextercete at gmail.com> wrote:
> I have made a translation from scratch, but now I want to update my .po file
> with the new strings created by stonecrest. I know it can be done, because I
> once saw a command in the Emesene forum, but I can't find it anywhere!

I believe that you're looking for msgmerge. So, if you have your own
translation file and there is a newer messages.po, you can:
msgmerge -U mine.po messages.po

If you have checked out the svn version of Sonata, you can simply run
the update-po script (thanks Mike!), and it'll update all translations
in the /po directory.
